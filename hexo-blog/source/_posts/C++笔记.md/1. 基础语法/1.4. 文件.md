# 1.4. 文件

# 文件

写入文件：

```cpp
#include <fstream>

ofstream file("test.txt");		// 创建输出流对象
file << "hello";				// 将文本写入文件
file.close();					// 关闭文件流
```

读取文件：

```cpp
#include <fstream>

string text;					// 用于存储读取内容
ifstream file("test.txt");		// 创建输入流对象
getline(file, text);			// 读取一行，并写入变量
file.close();					// 关闭文件流
```

读取多行：

```cpp
#include <fstream>

string text;
ifstream file("test.txt");
while(getline(file, text)) {
    cout << text << endl;
}
file.close();
```
