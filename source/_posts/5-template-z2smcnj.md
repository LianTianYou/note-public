# 5. 模板

泛型编程是 C++ 的编程思想之一，其主要的技术就是模板。
模板的分类：函数模板、类模板。

# 函数模板

定义函数模板：

- template：声明模板
- typename：定义类型参数，也可以用 class 代替。

```cpp
template<typename T>
函数声明或定义
```
